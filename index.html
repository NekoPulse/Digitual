Información Sobre Digitual

Digitual es un script para cualquier navegador y cualquier extensión que te permite acceder a cualquier sitio web, ya sea con Wifi público o privado, podrás acceder a cualquier página web.

Este script fue publicado el 12 de abril de 2025, cuenta con licencia MPL-2.0 y aplica a todos los sitios web y pesa alrededor de 83.3 KB, espero que lo disfruten.

¿Cómo instalar Digitual?

A continuación te explicaré paso a paso de como instalar Digitual en tu navegador.

Requisitos:

  • Tener computador
  • Alguna extensión instalada como Tampermonkey o ViolentMonkey.

Paso 1

Ir a la página web de Greasy Fork donde se encuentra Digitual. Click aquí

Paso 2

Instalar el script con su última versión, para esto solo le tienes que dar a el botón "install this script" o "instalar este script".

Paso 3

Y pues prácticamente ya completaste todo, podrás disfrutar de las funciones y esperar una nueva actualización :D
